Your interest motivated me to ship a new version of pyoptiwize.

For those just joining: pyoptiwize is a Python optimization cookbook covering real-world problems — production planning, scheduling, resource allocation and more.

The cookbook supports a wide range of solvers and algorithms:
🔹 SciPy (SHGO, Dual Annealing, SLSQP)
🔹 pymoo / NSGA-II & NSGA-III (multi-objective genetic algorithms)
🔹 DEAP (eaSimple genetic algorithm)
🔹 Pyomo — Python package for formulating and analyzing optimization models
🔹 SCIP — one of the fastest non-commercial solvers for mixed integer programming
🔹 IPOPT (Interior Point Optimizer) — open-source library for large-scale, continuous, nonlinear optimization
🔹 GLPK (GNU Linear Programming Kit) — open-source solver for large-scale LP and MIP problems
🔹 Optuna GP Sampler
🔹 Z3 SMT solver — standalone executable and called through PySMT API
🔹 Keras/TensorFlow, PyTorch & ONNX for black-box optimization

What's new in this release:

📦 One-script system dependencies install
A single install.bash replaces the old multi-step environment setup.

📓 Jupyter notebooks
Every example is now available as an interactive notebook. Explore, tweak parameters, re-run cells.

🤖 AI-Powered Problem Formulator
Describe your optimization challenge in plain English — get a solver-ready formulation back.
